Esse repositório contém o código solicitado para teste que recebe um valor em reais do usuário e converte para o mesmo valor por extenso.

Exemplo:
  input = R$25,50
  output = vinte e cinco reais e cinquenta centavos
  
  
     Para usar este código, basta clonar este repositório em sua maquina local, ou baixar o arquivo .ZIP. Abre em seu editor e execute o comando "npm install" no terminal para instalar todas as depedências necessárias. 
    Feito isso, basta rodar o código no terminal usando o comando "node index.js". O terminal solicitará que você insira um valor monetário. Digite no padrão internacional (usando vírgula no lugar de ponto decimal). O terminal irá retornar o valor inserido escrito por extenso.
